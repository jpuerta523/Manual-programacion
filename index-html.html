Diccionario Programación: HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos, es decir forma la estructura básica de los objetos que conforman una pagina web. La ultima versión es el estandar HTML:5


Estructura básica

Todas las páginas web pueden tener 4 partes principales en su estructura con caracteristicas definidas, a continuación se presenta una representación gráfica con sus respectivas etiquetas HTML, se puede crear la estructura báscia en el archivo HTML con el atajo ! + TAB:


<head > Elemento que contiene información util para el navegador pero que no es visible para el usuario
<body > Contiene los elementos con los que el usuario puede interactuar en la estrucutra principal del sitio.
<aside >
<footer> Muestra información util en el pie de página.

Ejemplo:


<!DOCTYPE html>
<html lang="es">
<head>
información util para el navegador como etiquetas meta, titulo y enlaces a otros archivos.
</head>
<body>
Cuerpo principal del sitio
</body>
<aside>
barras con información util laterales
</aside>
<footer>
Pie de página
</footer>
</html>


Etiquetas del head

Las etiquetas ubicadas antes o al interior del head, también conocidas como metainformación aporta información a los buscadores sin que esta sea visible a los usuarios por pantalla.


< !DOCTYPE html > Debe ser el primer elemento en cualquier proyecto HTML5, su función es declarar al navegador que es el archivo es de tipo HTML y se ubica en la primera linea. antes de definir el head, no tiene propiedades disponibles.


< html lang="es" > etiqueta raiz del documento HTML, abre después de la etiqueta < !DOCTYPE html > y cierra en la última línea del documento, tienen la siguiente propiedad: lang="es", ejemplos:


define el conjunto de caracteres disponibles en el lenguaje español: <meta charset="utf-8">
Indicar que es compatible con cualquer resolución de pantalla: <meta name="viewport" content="width=device-width, initial-scale=1.0" />
enlaza el archivo de estilo CSS: <link rel="stylesheet" href="estilo.css"/>
<title> Titulo mostrado en el navegador </title>


Etiquetas del DOM


Estas etiquetas definen la forma de como se le información al usuraio en pantalla ya sea texto, multimedia o hipervinculos.


Comentar texto

<!--> Etiqueta usada para comentar un texto en el código, esta información no es visible para el usuario en el navegador y usualmente se utiliza para dejar pista a los desarolladores, ejemplo:

<!-->Texto Comentado en el interior del código-->


Enlaces

<a> del anchor en íngles se utiliza para crear enlaces ya sea internos a otras secciones o a otras paginas en HTML, el atributo href="" define la ubicación del enlace y target="_blank", ejemplos:

Link a una parte en la misma pagina con ID de una etiqueta: <a href="#idSeccion">nombre de la sección del sitio </a>

Etiqueta de hipervinculo global, tagert blank abre el vinculo en una nueva pestaña: <a href="https://www.google.com/" target="_blank">haz click aqui </a>

Hipervinculo ubicado en una imagen: <a href="dirección hipervinculo"><img src="ubicación de imagen"> </a>

Etiqueta de hipervinculo a un archivo ubicado en una carpeta local: < a href="formulario plantilla.html" target="_blank">haz click aqui para abrir un pagina local </a>

Etiqueta para un hipervinculo local en carpetas: < a href="prueba/otra/formulario plantilla.html" target="_blank">haz click aqui para abrir un pagina local </a>

Etiqueta de hipervinculo local para volver a carpeta de orden superior: < a href="../formulario plantilla.html" target="_blank">haz click aqui para abrir un pagina local </a>

para enviar un correo: <a href="mailto:user@gmail.com">Contacto</a>

Formularios

<button> Crea un botón que al ser presionado por el usuario envia los cambios al servidor, en el atributo type tiene 3 opciones: button (para ejecutar codigo JS), submit(envia formulario a servidor) y reset(limpia info en el formulario)

<button type="submit"> Enviar </button>


<form> Indica que se va a crear un formulario en el documento HTML, todos los elementos del formulario son hijos de esta etiqueta, es decir estan en su interior tiene como atributo action que indica a donde se envia la información recolectada en el formulario y method indica el protocolo de envio.

<form action="Destino información formulario" method="post" >


</form>

Atributos

Los Atributos son las propiedades que pueden tener las etiquetas HTML

ID

Los ID son identificadores de elementos que asigna un nombre único que no debe repetirse en todo el documento

<p id="Nombre único">Texto de prueba</p>


Class

Permite agrupar bajo un mismo nombre de clase a varios elementos, muy util para definir reglas de estilo CSS, un elemento puede tener varias clases, solo hay que separarlas con un espacio.

<p class="clase1 clase2">Texto de prueba</p>


lang: Define el idioma del sitio, usada por la etiqueta < html lang="es" >
