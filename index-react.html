JSX

permite describir en JS como se verian los componetes de una pagina usando una estructura similar al HTML.

elementos

Es la unidad mas pequeña definida en react que define lo que se presenta en panatalla, con JSX puedes crear y usar en JS cualquier elemento definido como una etiqueta HTML en letras minusculas

Atributos

En JSX se pueden cambiar o asignar atributos CSS con un formato similar, a los atributos cuyos nombres contienen mas de 2 palabras separadas por un guión en CSS se les debe aplicar el concepto de camelCase para definirlos en JSX, algunos de sus nombres cambian por ser una palabra reservada . ejemplo:

  • cambio de nombre propiedad con camelCase:

    < h1 fontFamily = "arial"> </h1>

  • definir atributo Clase:

    < h1 className = "tituloClase"> </h1>

  • definir atributo for:

    < label HtmlFor = "tituloLabel"> tituloLabel </label>


Definir un estilo

para definir un estilo que agrupe varias reglas CSS en JSX, primero se defineuna variable const con el nombre del estilo y a continuación se crea un objeto JS donde cada una de sus propiedades corresponde a una propiedade CSS, ejemplo:

const estilo1 = { color: "red", fontFamily: "Arial", FontSize: 15 } Para asginar el estilo a un elemento JSX: <h1 style = {estilo1}>Titulo 1 </h1>

Crear una aplicación

pasos previos

Se necesita realizar una serie de pasos previos para iniciar una apliacion React

  • crear una nueva carpeta donde se aloja todo el proyecto y luego en el vsc se abre dicha carpeta con la ruta Archivo -> Abrir Carpeta.
  • Abrir la terminal integrada a VSC, se puede utilizar el atajo ctr + ñ, luego se verifica si la dirrección mostrada corresponde a la carpeta del proyecto, si ser requiere cambar de carpeta se utiliza el commando cd y luego la dirreción requerida.

  • en la terminal se ejecuta el siguiente codigo, donde la ultima palabara corresponde al nombre de la aplicación.

    npx create-react-app mi-app

    Este proceso inicia un proceso de instalación de los paquetes de React, usualmente tarda un par de minutos, al finalizar correctamente sale el aviso de Happy hacking! PS C:\Users\Puerta\Desktop\desarrollo\react bootcamp\carpeta del proyecto>.

  • se procede de nuevo a abrir la carpeta que contiene los nuevos archivos del proyecto en VSC, en el explorador de vcs se deben visualizar toda la nueva estructrura.

  • En la terminal se ejecuta el codigo npm start, lo que eventualmente en el navegador predeterminado abre una página con la dirrección localhost:3000, esto significa que nuestro equipo actua como un servidor local que permite visulizar los archivos react.

Mostrar un componente en la aplicación

todo el contendo de se agrgea entonces al interior de la etiqueta div con className app del archivo App.js, para ageragar un componete este va al interior de <> y su nombre debe comenzar con una letra mayuscula y este debe ser importado con su respectivo comando

import './App.css';
import Testimonio from "./componentes/Testimonio";

function App() {
return (
<div className="App">
<Testimonio />
</div>
);
}

export default App;

Asignar estilos CSS

Los estilos se asignan mediante import"..." a sus respectivos archivos, para la pagina principal se define las reglas CSS en el archivo App.css que viene por defecto en React, para los componenetes se recomienda crear un carpeta de estilos donde cada componete tiene su propio archivo CSS con nombres iguales.

Asignar props a un componente

para poder asignar props a un componente previamente creado se debe primero asignar props como variable en la función y en los compoenetes las propiedades dinamicas se marcan con la notación {props.propiedad}, si se busca un nombre dinamico en las propiedades como por ejemplo en las fotos la dirrección estaria adentro de backticks y la parte dinamica del nombre va al interior de ${}. ejemplo:

import React from "react";
import "../estilos/Testimonio.css"

function Testimonio(props){
return(
<div className="contenedor-testimonio">
<img
className = "imagen-testimonio"
src = {require(`../imagenes/testimonio-${props.imagen}.png`)}
alt = "foto de Emma"/>
<div className="contenedor-texto-testimoinio">
<p className="nombre-testimonio">{props.nombre} en {props.pais}</p>
<p className="cargo-testimoinio">{props.cargo} en {props.empresa}</p>
<p className="texto-testimonio">{props.testimonio}</p>
</div>

</div>
);
}

export default Testimonio;
