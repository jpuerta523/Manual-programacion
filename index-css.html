Diccionario Programación: CSS

CSS o Cascading style sheet es un lenguaje que aporta el estilo a un sitio web modificando los atributos a un archivo HTML existente


Definir reglas CSS


Estilo en linea:

Asigna atributos de estilo directamente en el elemento en el archivo HTML, ejemplo:

<p style="color: blue">Texto de prueba <p>

Texto de prueba


Elemento style:

Los estilos de los elementos de la página web se definen al interior de una etiqueta style ubicada al interior de la etiqueta del head, ejemplo:

...<head>

<style>

h4{

color: green;

}


p{

color: blue;

}

</style>

</head>

<body>...

<h4>Titulo de prueba<h4>

<p>Texto de prueba</p>

</body>...


Archivo CSS

Se recomienda por facilidades de lectura, mantenimiento y desarrollo ubicar todos los estilos de una página web al interior de un archivo CSS y luego vincularlo al archivo HTML que contiene la estructura del sitio, ejemplo:

en el archivo style.css

h4{

color: green;

}


p{

color: blue;

}

en el archivo index.html

...<head>

<link rel="stylesheet" href="estilo.css"/>

</head>...

<body>...

<h4>Titulo de prueba<h4>

<p>Texto de prueba</p>

</body>...


Selectores

Selector Universal

con el selector * se aplica las reglas CSS a todos los elementos del sitio que responden a dicha regla a no ser que esta propiedad se defina a un elemento con otro tipo de selector, ejemplo:

en el archivo style.css

*{

color: blue;

}

en el archivo index.html

<h2>Titulo de prueba</h2>

<p>Texto de prueba</p>


Elementos

Aplica las reglas CSS a todos los elementos que son del mismo tipo, ejemplo:

en el archivo style.css

p{

color: blue;

}

en el archivo index.html

<p>Texto de prueba</p>


Clases

.elemento: Aplica las reglas CSS a todos los elementos que tienen el mismo atributro de clase, ejemplo:

en el archivo style.css

.clase1{

color: blue;

}

en el archivo index.html

<p class="clase1">Texto de prueba</p>


Elementos hijos

elemento:nth-child(valor): permite aplicar reglas CSS a elementos hijos de un contenedor donde elemento es el contenedor y valor su ubicación,ejemplo:

en una tabla para selecionar las filas impares

tr:nth-child(even){

background-color: #fff

}

Id

#elemento: Aplica las reglas CSS a un elemento úncio definido por un ID de HTML, ejemplo:

en el archivo style.css

#nombre1{

color: blue;

}

en el archivo index.html

<p id=nombre1">Texto de prueba</p>



Varios elementos

Es posible agregar una regla CSS que aplique a varios elementos, simplemente se separan los elementos por una coma, ejemplo:

en el archivo style.css

h2,

p{

color: blue;

}

en el archivo index.html

<h2>Titulo de prueba</h2>

<p>Texto de prueba</p>


Selectores por atributo

Es posible agregar una regla CSS que aplique a un grupo de elementos que tienen un atributo en común, para ello se define el selector como el elemento y entre [] el atributo igual, ejemplo:

en el archivo style.css

input[type="checkbox"]

p{

margin: 5px;

}

selectores de hijos

Para un elemento contenedor es posible definir una regla CSS que afecten solo a los elementos hijos de un tipo definido como elemento o clase, solo es necesario nombrar primero al elemento contenedor y luego separado por un espacio definir los elementos a los que se le aplicara la regla, ejemplo

en el archivo style.css

div p{

margin: 5px;

}

las listas tambien tienen propiedades que pueden se modificada a traves de las reglas CCS

Estado del enlace

Se puede definir propiedades del enlace en los 4 estados que se definen a continuación, es importante respetar siempre el orden:

link: define las propiedades del enlace cuando este aun no ha sido visitado

visited: define las propiedades del enlace cuando el enlace ya fue visitado.

hover: define las propiedades del enlace cuando el cursor esta sobre el enlace.

active: define las propiedades del enlace en el momento que se esta dando click

en el archivo style.css

.enlace:link{
color:rgb(50, 25, 195);
font-weight: 700;
}

.enlace:visited{
color: rgb(4, 75, 4);
}

.enlace:hover{
color:rgb(221, 228, 12);
font-weight: 700;
}

.enlace:active{
color: rgb(191, 21, 106);
}

Tipo de fuente

font-family: Define el estilo de caracteres de un texto según unas familias predefinidas, en caso de que el navegador no reconozca la fuente se pueden poner varias como respaldo separadas por una espacio y una coma. tambien se pueden importar una fuente desde google fonts, ejemplo:

en el archivo style.css

p{

font-family: Impact, "times new roman";

}

en el archivo index.html

<p>Texto de prueba</p>

en el archivo index.html

<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

...<body>

Propiedades de elementos


Color

HEX: código hexadecimal con valores desde 0-9 y a-f iniciando con # y lo seguido de los dos primeros numeros que correspondem a la cantidad de rojo, el segundo a verde y el tercero a azul, ejemplo:#0000ff(azul)

HEX abreviado: similar a HEX pero solo con un digito de valor para cada color, ejemplo:#0f0 (verde>

RGB: o red-green-blue cada donde color puede tener un valor entre 0 y 255 y separados entre si por una coma, ejemplo: rgb(255,0,0) (Rojo)

RGBA: similar a RGB pero añade un cuarto valor alpha que define la transparencia del elemento al que se aplica el color, el valor a de 0 que es totalmente transparente a 1 que es opaco. ejemplo: rgba(120,120,0,0.5)

height: 50px;
background: rgb(230, 33, 33);

}

en el archivo index.html

Variables

Permite definir un valor como un propiedad que afecta varias reglas CSS, como principal ventaja tiene que en un solo cambio de variable actualiza los todos los elementos involucrados facilitando el proceso de diseño y mantenimiento de la página. opcionalmente se puede poner un valor de respaldo en las variables. para definir una variable como global se utiliza las pseudoclase :root{--variable:valor}. ejemplo

en el archivo style.css

div{
--nombre-variable: #00f
}
h2 {
color: var(--nombre-variable);
}
p {
color: var(--nombre-variable, blue);
}
en el archivo index.html

<div>

<h2>Titulo de prueba</h2>

<p>Parrafo de prueba</p>

</div>

Diseño Responsivo

@media. permite que las reglas CSS de diseño se adapten segun la resolución del dispositivo que visulaiza el sitio, ejemplo:

@media (max-width: 1199px) and (min-width: 769px) {
*{
color: #2f2;
}
