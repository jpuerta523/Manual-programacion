Diccionario Programación: JavaScript

JavaScript o JS es un lenguaje de programación debil, orientado a objetosy de tipado debil que se usa comunmente en el desarrollo Web y frontEnd.


Manejo Básico JS

JavaScript permite modificar y realizar acciones en un HTML y sus reglas CSS


Enlazar JavaScript con HTML

<script src=" ">...</script>: permite enlazar un archivo JS al interior de una estructura HTML, por regla general se ubica la etiqueta script al final del body para que se carge primero todo el contenido de la página y luego se pueda modificar con el JS, ejemplo:

en el archivo HTML ....
<body>

</body>


Ventanas Emergentes

Permite obtener o mostrar información al usuario, existen 3 tipos de ventanas

  • alert(): muestra una ventana con información al usuario, solo posee un botón para salir de la ventana emergente.
    en el archivo JS:
    alert("información Importante");

  • confirm(): Similar a alert con la diderencia que este posee 2 botones: uno para aceptar y otro para cancelar

    en el archivo JS:
    confrim("seguro que quieres salir de la página?");

  • prompt(): genera una ventana emergente con un cuadro de texto para que el usurio ingrese información

    en el archivo JS:
    let nombre = prompt("Ingrese el nombre del usuario:");


Consola

Permite verificar en la consola del navegador el resultado total o parcial de los resultado obtenidos en el codigo JS, existen varios metodos

  • console.log(): Muesta en la consola lo que se encuentre al interior de los parentesis, ya se una cadena de texto, el valor de una variable o el tipo de error si este existe.

    en el archivo JS:
    console.log("Cadena de texto en consola");

  • console.table(): muestra la información en la consola en formato tabla, especialmente util si la información mostrada es un arreglo

    en el archivo JS
    let arreglo = ["a", "b", "c", "d" ];
    en el archivo JS:
    console.log("Cadena de texto en consola");


  • console.error(): muestra un mensaje en consola como un error:

    en el archivo JS:
    console.log("Error: Cadena de texto en consola");

  • console.warn(): muestra un mensaje en consola como una advertencia:

    en el archivo JS:
    console.warn("Advertencia: Cadena de texto en consola");

t

let a = JavaScript;

let n = 6

Arreglos (arrays)

Arreglos simples

los arreglos son una agrupación de datos que permite alojar varios valores en una misma estructura. se define igual que una variable y los datos van al interior de un corchete [] y los datos separador por una coma, ejemplo:

let arreglo=["juan", "pablo", 36, "Medellín"];


Arreglos anidados

Un arreglo anidado es una agrupación de arreglos separados por una coma, ejemplo:.

let arreglo=[["bmw", "325i", 1992,], ["vw", "golf", 2005], ["suzuki","swift", 2019]];


Longitud de un arreglo:

arr.length: Permite saber cuantos elementos posee un arreglo, ejemplo:

let arreglo = ["bmw", "325i", 1992] ;
console.log(arreglo.length); resultado en consola: 3


Acceder elementos

Es posible acceder a los elementos al interior de un arreglo para posteriormente poder manipularlos, para ello se utiliza la misma sintaxis con corchetes [] utilizadas para llamar los caracteres en una cadena de texto enpenzando en el indice 0 para el primer elemento, ejemplo:

para la primera posición:

let arreglo=["juan", "pablo", 36, "Medellín"];

let a = arreglo[0];

console.log(a);

resultado en consola:

"juan"


para la posición n:

let a = arreglo[2];

console.log(a);

resultado en consola:

36



Modificar Arreglos

Es posible cambiar un valor que este al interior de un arreglo, simplemente se accede al arreglo con notación de corchetes [] y se le asigna el nuevo valor con un =, si es un arreglo anidado se usan doble notación de corchete donde el primero ubica la posición en el arreglo padre y el segundo en el arreglo anidado, si la posicion modificada no existe en el arreglo original, esta se crea y las posiciones anteriores y superiores a los indices origineales se crean como elementos vacios. ejemplo:

para la primera posición:

let arreglo=[1, 2, 3, 4 ];

arreglo[0] = 0 ;

console.log(arreglo);

resultado en consola:

[0, 2, 3, 4 ]

let arreglo=[1, 2, 3, 4 ];

arreglo[5] = 10;
console.log(arreglo);
resultado en consola:
[0, 2, 3, 4,,10 ]


para un arreglo anidado segundo arreglo y tercer elemento:

let arreglo=[[1, 2, 3], [4, 5, 6], [7, 8, 9]];

a =arreglo[1][2] ;

console.log(a);

resultado en consola:

6

Recorrer el DOM

Algunas veces se requiere conocer los elementos que estan enparentados con un elemento especifico ya sea a nivel de hermanos, padres o hijos. ejemplo:

  • let a = $elemento.previousElementSibling: devuelve el hermano ubicado inmediantamente antes del elemento
  • let a = $elemento.nextElementSibling: devuelve el hermano ubicado inmediantamente despues del elemento
  • let a = $elemento.parentElement: devuelve al padre o contenedor del elemento
  • let a = $elemento.parentElement.parentElement: devuelve al abuelo o contenedor del contenedor del elemento
  • let a = $elemento.Child: devuelve un HTML colection o listado con los hijos del elemento
  • let a = $elemento.lastElementChild: devuelve al último hijo del elemento
  • let a = $elemento.firstElementChild: devuelve al primer hijo del elemento
  • let a = $elemento.childElementCount: devuelve el dato de la cantidad de elementos hijos que contiene.
