Diccionario Programación: GIT

GIT es un programa de desarrollo abierto que permite controlar las versiones de control de los proyectos.

Instalación

Para instalara GIT en tu computadora puedes acceder al sitio oficial de GIT y escoje la opción de descarga según tu sistema operativo. para windows se descarga GIT Bash que es la consola con la que se trabaja a continuación.

Configuración de la terminal

configuracion global GIT

permite definir un nombre de usuario y correro con "git config --global usermane "nombre"

Abrir carpeta proyecto

al abrir la carpeta que contiene el proyecto, es posible si git esta instalado abrir directamente en esa ubicación, para ello dando click derecho se busca la opcion "git Bash here", esto abre la carpeta en la terminal Bash

Inicializar repositorio

Para inicializar el repositorio de nuestro archivo, solo es necesario usar el comando "git init"

$ git init


Validar archivos en GitBash

para verificar que archivos estan cargados en GIT, en la terminal se utiliza el comando "git status -s" que como resultado da un listado con todos los archivos, inicialmente todos los archivos van con un par de interrogaciones (??) en color rojo indicando que los archivos son visibles para GIT pero no les esta realizando ningun seguimiento o acción.

$ git status -s


Seguimiento a un archivo particular

Si se buscar hacer seguimiento en Git de un archivo en especifico se utiliza el comando "git add archivo.extencion" donde archivo.extencion es el nombre del archivo y su correspondiente extensión, en el listado de estatus el archivo aparace con una "A" en color verde o area de ensayo.

$ git add index.html


Enviar al repositorio

Para enviar todos los archivos que se encuentren en seguimiento al repositorio se usa el comando "git commit -m "descripción", ya estos archivos no son visibles en el listado de estado

$ git commit -m "descripción"


Guardar cambios en un archivo

Si posterior al envio al repositorio, un archivo es modificado y se guarda los cambios, este en la lista de estado aparece de nuevo con una letra "M" en color rojo indicando que el archivo fue modificado, posteriormente se debe enviar a la zona de seguimiento con el comando "git add archivo.extencion" quedando en la lista de estado con la "M" en verde y finalmente se envia al repositorio con el comando "git commit -m "descripción"

$ git add index.html $ git commit -m "Se agrega mas contenido a la página"


Revisar repositorios

Para revisar cuantos repositorios o versiones se han guardado con exito se utiliza el comado "git log --oneline" desplegando el listado con sus correspondientes comentarios

$ git log --oneline"


Restaurar un archivo a una version anterior

para restaurar un archivo a una versión previa guardada en un repositorio es necesario utilizar el comando "git reset -hard codigo" donde codigo es una combinacion alfanumerica unica para cada archivo asignado en la lista de estado

$ git reset --hard 76a6140"
